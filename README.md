# QA-CYPRESS

For open the Cypress App execute the command:
```
npx cypress open
```
To run tests
```
npx cypress run
```

# To start at GIT, please follow this steps:

Start at main branch:
```
git checkout main
```
Get all updates:
```
git pull
```
Create a new branch named as the same as issue(Jira):
```
git checkout -b '<JIRA ISSUE>'
```
